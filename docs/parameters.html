Frequency grid parameters ///.

KATANIN

If defined, the Katanin extension is used during mfRG computations.

KELDYSH_FORMALISM

Determines whether calculations shall be done in the Keldysh or Matsubara formalism.

0 for Matsubara; 1 for Keldysh formalism.

MAX_DIAG_CLASS

Production runs parameters ///.

Defines the diagrammatic classes that will be considered: 1 for only K1, 2 for K1 and K2 and 3 for the full dependencies. Useful for debugging purposes and for computations in second order perturbation theory, which only require K1.
