template<typename Q >
auto correctionFunctionSelfEnergy_Keldysh ( int iK,
double vmin,
double vmax,
Q eps_p,
double Delta,
char type ) -> Q

Helper function for computing the analytical result for the asymptotic tails of the loop integral, assuming the self-energy of the propagator to be decayed to the Hartree value. See the function asymp_corrections_loop below.

Parameters
iK: Type of the loop propagator 0: G^R, 1: G^A, 2: G^K
vmin: Lower limit of the numerically evaluated integral
vmax: Upper limit of the numerically evaluated integral
Sigma_H: Hartree self-energy
Delta: Hybridization (~ flow parameter) at which the bubble is evaluated
type: Propagator type ('g': full, 's': single-scale)
Returns
: Analytical result of the integrated tails
