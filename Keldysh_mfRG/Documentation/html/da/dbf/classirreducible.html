Detailed Description

template<class Q>
class irreducible< Q >

Reducible vertex in channel r, split into diagrammatic classes K1, K2, K2b, K3. Member functions allow for access of values at arbitrary frequencies and in each channel-specific frequency representation. Possible (unit-)tests: [IMPLEMENTED in test_symmetries.c++] check read-out from symmetry-reduced sector and correctness of symmetry tables [MISSING] check conversion of frequency conventions

Possible unit-tests Do the public functions return the correct vertex-contributions?
