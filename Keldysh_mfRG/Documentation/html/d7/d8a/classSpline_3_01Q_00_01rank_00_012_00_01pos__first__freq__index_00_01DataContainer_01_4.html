template<typename Q, size_t rank, my_index_t pos_first_freq_index, class DataContainer>
class Spline< Q, rank, 2, pos_first_freq_index, DataContainer >

SplineK2 interpolation

Template Parameters
ranknumber of dimensions (of dataContainer)
pos_first_freq_indexposition of first frequency index (second frequency index is as position pos_first_freq_index+1)
DataContainercontains vertex data and frequency grids frequencies. primary_grid and frequencies.secondary_grid computes partial derivative of data in x, y and x&y direction
Qdouble or comp
