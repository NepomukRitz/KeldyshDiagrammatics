inline

This file contains all information of the symmetry-transformations table:

  • The struct Components contains the information to which independent Keldysh components all other Keldysh components in each diagrammatic class K1, K2, K2b, K3 are related, and
  • the struct Transformations contains the information how this relation is achieved, i.e., which transformations have to be applied to access a stored Keldysh component symmetry-related to the desired one.
