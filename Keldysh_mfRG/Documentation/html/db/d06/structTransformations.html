Detailed Description

Transformations that need to be applied to the respective stored components to get the correct actual components: 0 = nothing, 1 = T1, 2 = T2, 3 = T3, 4 = TC Convention for composite trafos: 43 = first apply 4, then 3 etc. Careful, some operations do not commute!
