Detailed Description

template<typename Q, typename vertType, bool all_spins, typename return_type, bool version>
class IntegrandSE< Q, vertType, all_spins, return_type, version >

Class for the integrand of the Retarded SelfEnergy Requires a fullvertex (ref), a propagator(ref), an input frequency and an internal structure index

Template Parameters
QType in which the integrand takes values, usually comp Class for the integrand of a loop calculation. Invoked by LoopCalculator.
QType of the data.
vertTypeType of the vertex to be used.
all_spinsCompute all spin components?
return_typeType of the data of the result. Typically = Q.
versionVersion of the self-energy loop. See documentation for LoopFunctionCalculator.
