Detailed Description

template<typename Q, typename vertType, bool all_spins, bool version>
class LoopCalculator< Q, vertType, all_spins, version >

Class to actually calculate the loop integral for a given external fermionic frequency and internal index.

Self-energy loop The selfenergy loop can be closed in two different versions: version 0: close the loop above with propagator G^{3|1} => parametrize vertex in t-channel convention with w=0 version 1: close the loop right with propagator G^{2|1} => parametrize vertex in a-channel convention with w=0 3 -> _____ -> 1 | | 0 <-|____|<- 2 Note: the vertices in versions 0 and 1 are related by crossing symmetry, hence we expect the results to be identical in both versions
