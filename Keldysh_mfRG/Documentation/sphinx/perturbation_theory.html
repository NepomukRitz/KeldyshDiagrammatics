Template Parameters:

Q – Data type of the state, usually comp

Parameters:

  • Psi – State whose Vertex is to be calculated

  • Bubble_Object – Previously initialized bubble object.

  • Lambda – Data structure-needed parameter. Should be set to 1. in all SOPT calculations

  • state – State whose Vertex whould be the bare vertex already initialized
