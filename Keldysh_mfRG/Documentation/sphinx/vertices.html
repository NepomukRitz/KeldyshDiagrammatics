template<class Q>
class irreducible

Reducible vertex in channel r, split into diagrammatic classes K1, K2, K2b, K3. Member functions allow for access of values at arbitrary frequencies and in each channel-specific frequency representation. template<class Q>
class fullvert

The class fullvert The class defining a vertex with full channel decomposition i.e. irreducible (bare) a, p and t channels The fullvert in the standard configuration is the full vertex \Gamma with all components (only_same_channel=false and Ir=false) for Ir = true it is an r-irreducible vertex I_r (r is the channel of the bubble) for only_same_channel it only contains the r-reducible component

class GeneralVertex

symmetric_full and symmetric_r_irred vertex container class (contains only half 1, since half 1 and 2 are related by symmetry) non_symmetric contains the actual vertex in “vertex” and the left-right-symmetry-related one in “vertex_half2” (the latter is needed for reading from the symmetry-reduced sector)
