State

template<typename Q, bool differentiated = false>
class State

Define a struct object which includes the self-energy and the vertex which are needed to evaluate the RHS of the flow equations.

Define essential data types: comp : complex number (complex<double>) glb_i : imaginary unit vec : vector class with additional functionality such as element-wise operations, real/imag part etc. VertexInput : auxiliary struct that contains all input variables of vertices

Functions to write/read a State object to/from an HDF5 file.
